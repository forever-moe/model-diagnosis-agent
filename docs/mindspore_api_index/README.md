# MindSpore API Index

This directory stores generated artifacts and companion documents for the
`mindspore.mint` API index.

## Official Builder

Use the official entrypoint below to regenerate all artifacts:

```powershell
python tools\mindspore_api_index\build_mint_index.py --repo code\mindspore --out docs\mindspore_api_index
```

`tools\mindspore_api_index\build_mint_index.py` is the only maintained builder
entrypoint.
